# Практическая работа

## Правила выполнения заданий

1. При решении всех заданий данной работы **требуется реализовать программы, эффективные по времени и памяти**. 
2. **Запрещено использовать динамические структуры данных**: списки (`list`), словари (`dict`), множества (`set`). Любое решение с использованием списков, множеств, словарей не будет засчитано!
3. Используйте переменные, циклы, условные операторы и встроенные функции, не создавая дополнительные массивы данных в памяти кроме служебных переменных.

### Задания

1. На вход подаются N целых чисел.
Выведите наименьшее нечётное число среди них. Если таковых нет — выведите «NO».

---

2. Вводится число N, затем N натуральных чисел.  
Посчитайте и выведите количество чисел, делящихся на 5, и их сумму.  
Если таких чисел нет — выведите «NO».

---

3. На вход поступает последовательность из N целых чисел.  
Определите наибольшее отрицательное число среди них.  
Если таковых нет — выведите «NO».

---

4. Вводится число N, затем N чисел.  
Найдите минимальное и максимальное число среди введённых и выведите их произведение.
